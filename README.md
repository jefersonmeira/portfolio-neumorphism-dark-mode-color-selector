## Portfólio responsivo em HTML5, CSS3 e JavaScript

_Projeto desenvolvido durante os estudos em HTML5, CSS3 e JavaScript._ 

### Neumorphism ~ Modo Noturno ~ Seleção de Cores

Projeto para criação de um Portfólio:
- Aplicado o conceito de NEUMORPHISM no projeto todo.
- Implementada a opção para ativar/desativar o modo noturno.
- Implementada a opção para selecionar uma cor de destaque do portfólio.
- O projeto é totalmente responsivo, se adequando aos diferentes tipos de tela.

Foram utilizadas poucas instruções JavaScript para:
- Apresentar e esconder o menu (em determinadas resoluções de tela);
- Ativar/desativar modo noturno;
- Abrir/Fechar paleta de cores;
- Ativar cores de destaque do portfólio.

A estrutura do projeto possui: 
- 1 menu de navegação; 
- 5 seções de conteúdo;
- 1 rodapé.
